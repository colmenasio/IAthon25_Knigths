# Datua Dataset Disclaimer

This dataset is composed of 2 separate csv files. We did not touch this dataset during the hackaton, so we didn't bother to implement the possibility to provide an arbitrary number of datasets to the isolated data analysis environment.

Although this feature would be extremely easy to implement, we consider doing so after the hackaton is already finished is a bit "cheaty" so the results of our analysis will be the ones obtained by running our data analysis software separately on both datasets

Analysis 1 corresponds to "TV Series Dataset"
Analysis 2 corresponds to "video streaming application"
