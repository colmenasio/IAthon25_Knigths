Detailed Analysis Report on Factors Affecting max_Z_wear

Introduction

The objective of this analysis was to investigate the dataset containing various parameters affecting the target Key Performance Indicator (KPI), max_Z_wear. The analysis utilized statistical methods to determine the significance of each parameter and to identify strategies to minimize the target KPI.

Dataset Overview

The dataset consists of 9068 entries with 13 columns. The columns include categorical variables, continuous variables, and the target variable. Below is a brief description of the dataset columns:

  • Case: Train model (e.g., "Model 1", "Model 2")
  • Load: Passenger load (e.g., "AW0" (TARE), "AW4" (MAX LOAD))
  • Configuration: Train type (e.g., "Conf1", "Conf2", "Hyb")
  • Wheel.Profile: Wheel profile design criterion (e.g., "P1", "P2")
  • COF: Coefficient of friction (Dry or with flange lubrication)
  • Track: Track geometry severity (1-Low, 2-Med, 3-High)
  • Direction: Travel direction (Unidirectional or Bidirectional)
  • F_KX1: Design variable 1 (Continuous)
  • F_KZ1: Design variable 2 (Continuous)
  • F_KZ2: Design variable 3 (Continuous)
  • F_S: Design variable 4 (Continuous)
  • max_Z_wear: Target KPI (to minimize, Continuous)

Data Exploration

Summary Statistics

The preliminary examination of the dataset was carried out using the describe and info methods from pandas. The dataset was found to have no missing values and included various types of data (categorical and continuous).

Correlation Analysis

A correlation matrix was computed to assess the relationships between the variables and max_Z_wear. Notably, the variables with the most significant correlations with max_Z_wear were: - COF: -0.605142 - Wheel.Profile: 0.469293 - Track: 0.299100 - Direction: 0.173086 - Case: 0.172672 - Configuration: -0.101949

ANOVA Analysis

ANOVA tests were conducted for categorical variables. The following results indicated statistical significance (p-value < 0.05): - Wheel.Profile: F-value = 2560.60, p-value = 0.00 - Track: F-value = 469.32, p-value = 1.13e-194 - Direction: F-value = 279.99, p-value = 6.36e-62 - Case: F-value = 278.61, p-value = 1.24e-61 - Configuration: F-value = 51.53, p-value = 5.56e-23 - COF: F-value = 5238.12, p-value = 0.00

Continuous Variable Analysis

Regression analysis was attempted to determine the relationship between the continuous variables (F_KX1, F_KZ1, F_KZ2, F_S) and max_Z_wear. However, the t-values for these variables indicated that none of them were statistically significant with respect to max_Z_wear.

Summary of Findings

  1. Significant Factors Affecting max_Z_wear:
  2. COF: Strong negative correlation; lower COF values reduce wear.
  3. Wheel.Profile: Profile P1 has lower wear than P2.
  4. Track: Lower severity tracks lead to reduced wear.
  5. Direction: Unidirectional travel reduces wear compared to bidirectional.
  6. Case: Model 1 has lower wear than Model 2.

  7. Configuration: Configuration 2 shows lower wear than other configurations.

  8. Non-significant Continuous Variables:

  9. None of the continuous design variables showed significant t-values, indicating they may not be effective in reducing max_Z_wear.

Recommendations

Based on the analysis, the following actions are recommended to minimize max_Z_wear: 1. Utilize COF 0 for operations to minimize wear. 2. Prefer Wheel Profile 0 to reduce wear. 3. Optimize operations on lower severity tracks (Track 0). 4. Consider unidirectional travel where feasible. 5. Favor Case 0 in planning operations. 6. Implement Configuration 2 for optimal performance.

Conclusion

The analysis highlights the significant factors affecting max_Z_wear and provides actionable insights for minimizing wear in train operations. By focusing on the identified significant variables and their optimal configurations, it is possible to achieve better performance and longevity of the train components.
